## What this site is

With Joyful Lips Lyrics is a fast, searchable reference site for hymn lyrics, prepared for real-world use in Catholic worship settings. The pages are formatted to be easy to read, easy to copy, and easy to paste into slides and handouts.

## Source and attribution

The lyrics on this site are taken from the Catholic hymnal *With Joyful Lips* by Fr. Jean L. Mercier (Twelfth Revised Edition). The underlying hymn texts remain the property of their respective rights holders.

This site is an independent project and is not affiliated with the author or publisher.

## Intended use

This site exists to support:

- Worship leaders preparing liturgies
- Choir members and musicians rehearsing
- Projection teams creating slides
- Personal prayer and study

If you need licensed, commercial use, please obtain the appropriate permissions from the rights holders.

## How to use the lyrics

1. Find a hymn by category or search for a title/phrase.
2. Copy the lyrics you need.
3. Paste as plain text (Ctrl+Shift+V / ⌘+Shift+V) to avoid unwanted formatting.
4. Keep attribution intact when appropriate.

## Accuracy and corrections

Lyrics are transcribed carefully, but mistakes can happen. If you spot an error (spelling, punctuation, missing verse, wrong heading), please report it through the project repository so it can be fixed for everyone.

## Privacy

This site is static and does not require accounts. Basic analytics may be used to understand overall usage (e.g., which pages are most visited) so the collection can be improved.
